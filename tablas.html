Atributos
atributo función
  • align
    		•  hace que el objeto flote hasta el margen izquierdo o derecho.
  • valign
    		•  Indica la alineación vertical de los datos dentro de las celdas.
  • background
    		•  definir los valores individuales del fondo en una única regla CSS.
  • bgcolor
    		•  es usado para determinar el color de fondo de un párrafo, tabla o cualquier otra parte del HTML
  • cellpadding
    		•  indica los píxels de separación entre el borde de la celda y su contenido.
  • cellspacing
    		•  Define el espacio entre los bordes (en pixels).
  • border
    		•  es una forma rápida para establecer el ancho, estilo y el color del borde izquierdo de un elemento.
  • height
    		•  especifica la altura de un elemento.
  • width
    		•  establece el ancho de un elemento mediante un valor numérico (con su unidad) o un porcentaje (con respecto al elemento padre).
